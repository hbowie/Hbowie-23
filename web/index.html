Built with Notenik

Available for free from the Mac App Store, Notenik can be used for taking, organizing, formatting and publishing notes, with each note consisting of multiple fields, as well as chunks of Markdown text of any length. The HBowie.net website has been built using Notenik.

Contents Outline:

This site’s content falls into a number of different categories. Feel free to drill down into any area that interests you. Stay connected through one or more of several avenues.

Hot Off The Press…

See a list of my latest original writings, as well as the latest quotes added to my commonplace book. Visit the Discourse Forum for the latest Notenik news.

Software

Notenik is my note-taking, web-making, content-managing app, written in Swift, using AppKit, exclusively for the Mac. Find out more at Notenik.app.

Big Ideas in Software Development is my web book about software development. You can find it at SoftDevBigIdeas.com.

Songs and Music

I’ve written many pieces discussing songs that I’ve found compelling in one way or another. I talk about their themes and meanings, as well as how they work.

  • Songs Index – A list of content on this site.
  • Lexicon of Song – A separate site offering this content.
  • Reason to Rock – A separate site offering an entire online book discussing the aesthetics of rock music.
